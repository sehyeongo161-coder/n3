# iPhone에 JLPT N3 단어 퀴즈 설치하기

이 폴더는 iPhone 홈 화면에 설치할 수 있는 PWA 구성입니다.

## 먼저 해야 할 일

`outputs` 폴더 안의 파일과 `icons` 폴더를 **구조 그대로 HTTPS 웹 호스팅에 업로드**합니다.

- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icons/` 폴더 전체

`file://`로 파일을 직접 여는 방식은 일반 웹페이지로는 작동하지만, 홈 화면 앱 설치와 오프라인 캐시는 사용할 수 없습니다.

## iPhone에서 설치

1. iPhone의 Safari에서 업로드한 HTTPS 주소를 엽니다.
2. 아래쪽의 **공유 버튼(□↑)** 을 누릅니다.
3. **홈 화면에 추가**를 선택합니다.
4. 오른쪽 위의 **추가**를 누릅니다.
5. 홈 화면의 **N3 단어** 아이콘을 눌러 실행합니다.

설치 후에는 Safari 주소창 없이 앱처럼 전체 화면으로 실행됩니다. 처음 한 번 온라인으로 연 뒤에는 앱 파일이 캐시에 저장되어 오프라인에서도 사용할 수 있습니다.

학습 기록과 오답은 iPhone의 해당 홈 화면 앱 저장소에 보관됩니다. Safari 데이터를 삭제하거나 앱을 제거하면 기록도 사라질 수 있습니다.
